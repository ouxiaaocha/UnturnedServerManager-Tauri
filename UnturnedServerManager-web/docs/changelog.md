# 更新日志

## v2.2.0

### 稳定性

- 修复长时间运行时前端事件监听器和页面定时器清理问题。
- 优化 RCON 响应缓冲区，限制单条响应长度，降低长期运行内存峰值。
- 调度器定期清理孤儿任务记录。
- 日志默认保留天数调整为更适合便携目录的策略。

### UI 与工作流

- 新增通用 Button、Card、Select、SelectCustom 组件。
- 重构存档页面为标签页工作台：基础配置、高级配置、工作坊、插件、权限。
- 新增 Config.txt 高级配置编辑器。
- 新增工作坊模组备注、插件备注和权限管理界面。
- 增强窗口管理：托盘菜单、关闭行为记忆、运行服务器退出保护。

### 安全与维护

- 强化日志路径校验，降低路径穿越风险。
- 优化子进程环境变量清理。
- 本地命令 Bridge 成为日常启停和重启主路径，RCON 保留为额外远程能力。

## v2.1.0

- 服务器控制主路径切换为本地命令 Bridge。
- 服务器页面新增本地命令输入栏。
- 设置页新增运行环境检测与修复入口。
- 更新检测同时显示本地版本与云端版本。
- 优化 SteamCMD 更新流程、自动重启日志续接和启动期间页面响应。

## v2.0.1

- 仪表盘新增运行时信息卡片。
- RCON 连接性能优化。

- 新增 VitePress 文档网站。

## v2.0.0

- 新增模组备注、插件备注功能。
