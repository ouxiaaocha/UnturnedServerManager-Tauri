# 📝 更新日志

## v2.1.0

### ✨ 新功能

- 服务器控制主路径切换为本地命令 Bridge，不走 RCON 即可发送控制台命令
- 服务器页面新增本地命令输入栏，发送成功后显示 `[命令] > ...`
- 仪表盘、服务器页的停止和重启改为本地命令优雅关闭
- 设置页新增运行环境检测和修复入口

### 🔧 改进

- 优化 SteamCMD 更新流程显示
- 软件更新检测同时显示本地版本和云端版本
- 优化启停期间 UI 响应
- 重启流程增加等待和日志续接
- 启动前强制校验 Rocket/Bridge 条件

### 🐛 修复

- 修复本地命令队列并发读写丢命令的问题
- 修复重启后日志不继续刷新的问题
- 修复快速检测未执行 SteamCMD 网络测试的问题
- 修复 GitHub Release Markdown HTML 注入风险
