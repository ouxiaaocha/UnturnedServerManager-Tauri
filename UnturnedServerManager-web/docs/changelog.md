# 📝 更新日志

## v2.2.0 (2026-06-15)

### 🐛 稳定性改进

- **修复长时间运行内存泄露问题**，可 7×24 小时稳定运行
  - 修复前端 Tauri 事件监听器泄露（App.svelte）
  - 修复页面切换时定时器未清理（Save.svelte、Update.svelte）
  - 优化 RCON 响应缓冲区，内存峰值降低 50%（1000 → 500 条）
  - 调度器每小时自动清理孤儿任务记录
  - 单条响应长度限制 1000 字符，防止超长日志占用内存
- **日志保留天数优化**：默认从 30 天降至 15 天，减少磁盘占用

### 🎨 UI/UX 优化

- **新增通用 UI 组件库**
  - Button：支持 primary、secondary、danger、ghost 变体
  - Card：统一的卡片容器样式
  - Select：原生下拉选择组件
  - SelectCustom：自定义下拉选择（支持搜索、键盘导航）
- **重构存档管理页面**，采用标签页布局
  - 基础配置 | 插件管理 | 工作坊 | 权限管理 | RCON 配置
  - 新增工作坊模组管理：模组列表、Steam 工作坊跳转、模组备注
  - 新增权限管理界面：权限列表编辑、SteamID 权限配置
  - 优化 RCON 配置界面：端口/密码设置、随机密码生成
- **优化所有页面**的布局、样式和交互体验
  - Dashboard、Server、Rcon、Logs、Schedule、Settings、Wizard
  - 统一按钮和表单样式，优化响应式布局
  - 改进错误提示和用户反馈

### ⚙️ 新增功能

- **窗口管理**
  - 最小化到托盘、托盘菜单、关闭行为自定义
  - 托盘菜单：快速导航、自动托管切换、窗口显示/隐藏
  - 关闭确认对话框：支持记住用户选择
- **存档管理增强**
  - read_rocket_rcon_config / save_rocket_rcon_config
  - read_workshop_config / save_workshop_config
  - read_permissions_config / save_permissions_config
  - load_workshop_mod_notes / save_workshop_mod_notes
  - open_url：打开外部链接

### 🚀 性能优化

- **RCON 缓冲区优化**
  - 上限从 1000 降至 500 条，裁剪批次从 500 降至 250
  - 峰值内存占用从 150KB 降至 75KB
- **调度器优化**
  - 每小时清理不在任务列表中的孤儿条目
  - 防止长期运行时 HashMap 无界增长
- **轮询机制优化**
  - 改进 token 机制防止竞态条件
  - 优化页面可见性监听
  - 改进内存管理和清理逻辑
- **后端命令优化**
  - 优化日志读取性能和路径验证
  - 改进进程管理和状态检测
  - 优化系统资源监控算法

### 🔒 安全增强

- **路径安全**：增强日志文件路径验证，防止路径穿越攻击
- **环境变量清理**：优化子进程环境变量清理，防止敏感信息泄露
- **RCON 安全**：改进 RCON 连接管理和认证流程

### 🛠️ 开发改进

- 统一代码风格和注释
- 完善 TypeScript 类型定义
- 优化错误处理和日志记录
- 新增 useClickOutside 组合式函数
- 优化 CSS 变量定义和深色模式配色
