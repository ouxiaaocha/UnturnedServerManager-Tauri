# 更新日志

## v2.1.0

### 新功能

- ✨ 服务器控制主路径切换为本地命令 Bridge，不走 RCON 即可发送 `Save`、`Say`、`Shutdown` 等控制台命令
- ✨ 服务器页面新增本地命令输入栏，发送成功后会在输出区显示 `[命令] > ...`
- ✨ 仪表盘、服务器页的停止和重启改为本地命令优雅关闭，RCON 仅作为额外远程功能保留
- ✨ 设置页新增运行环境检测，可检查 SteamCMD、服务端目录、Rocket.Unturned 和 `UnturnedServerManagerBridge.dll`
- ✨ 设置页新增修复入口，可重新部署 Rocket.Unturned 和本地命令 Bridge

### 改进

- 🔧 优化 SteamCMD 更新流程，显示 SteamCMD 自更新、服务端校验和最终安装状态
- 🔧 软件更新检测同时显示本地版本和云端版本，更新日志安全渲染
- 🔧 优化启动、停止、重启期间的 UI 响应，减少页面切换卡顿
- 🔧 重启流程增加等待和日志续接，降低旧进程未退出时重复启动的风险
- 🔧 启动服务器前强制校验 Rocket/Bridge 条件，缺失时给出明确错误

### 修复

- 🐛 修复本地命令队列并发读写时可能丢命令的问题
- 🐛 修复游戏重启后服务器日志不继续刷新的问题
- 🐛 修复快速检测里未执行 SteamCMD 网络测试却显示通过的问题
- 🐛 修复 GitHub Release Markdown 直接渲染带来的 HTML 注入风险
