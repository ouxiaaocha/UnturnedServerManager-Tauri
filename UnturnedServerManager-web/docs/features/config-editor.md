## 入口

进入「存档」，选择目标存档，然后切换到「高级配置」标签。该页面不是主导航一级页面，而是存档工作台的一部分。

## 能力

| 功能 | 说明 |
| --- | --- |
| 分组展示 | 按 Config.txt 的配置段落组织字段 |
| 搜索筛选 | 快速查找性能、规则、难度等配置项 |
| 状态标记 | 区分默认值、覆盖值和本次改动 |
| 安全保存 | 保存时带源文件哈希，降低覆盖外部改动的风险 |
| 运行保护 | 存档运行中时避免修改关键配置 |

## 与基础配置的关系

基础配置面向高频项，适合快速设置名称、地图、端口和 RCON；高级配置面向完整 Config.txt，适合细调性能、规则和游戏体验。

::: warning 注意
大多数 Config.txt 变更需要重启服务器才能生效。保存后请在「服务器」页重启对应存档。
:::
