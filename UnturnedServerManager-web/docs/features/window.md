窗口管理功能提供灵活的后台运行选项，支持最小化到系统托盘、托盘菜单快速导航，以及自定义关闭行为，让您在管理服务器时获得便捷的后台体验。

## 功能概览

| 功能 | 说明 |
| --- | --- |
| 最小化到托盘 | 点击最小化或关闭按钮时隐藏窗口，程序在后台继续运行 |
| 托盘菜单 | 右键点击托盘图标访问快速导航菜单 |
| 关闭行为自定义 | 选择关闭按钮的行为：关闭到托盘或完全退出 |
| 快速导航 | 从托盘菜单快速跳转到各功能页面 |
| 自动托管切换 | 直接从托盘菜单开启/关闭自动更新托管 |

## 托盘功能

### 最小化到托盘

当您需要让程序在后台运行而不占用任务栏空间时：

1. **最小化窗口**：
   - 点击窗口右上角的最小化按钮
   - 或者根据您的关闭行为设置，点击关闭按钮（❌）

2. **窗口隐藏**：
   - 窗口会立即隐藏到系统托盘
   - 程序继续在后台运行
   - 托盘图标显示程序状态

3. **恢复窗口**：
   - **单击托盘图标**：显示/隐藏主窗口
   - **从托盘菜单选择「显示窗口」**：显示主窗口

::: tip 提示
最小化到托盘后，服务器监控、定时任务、自动托管等功能会继续在后台运行，不受影响。
:::

### 托盘菜单

右键点击系统托盘中的程序图标，可以打开快速操作菜单：

**导航功能**：
- 仪表盘
- 服务器控制
- RCON 控制台
- 存档配置
- 定时任务
- 设置

**快速操作**：
- **显示窗口** / **隐藏窗口**：切换主窗口显示状态
- **自动托管**：开启/关闭服务端自动更新托管（显示当前状态）
- **退出**：完全关闭程序

::: tip 提示
托盘菜单提供了最常用功能的快速访问入口，无需打开主窗口即可快速导航。
:::

## 关闭行为设置

您可以自定义点击关闭按钮（❌）时的行为。

### 关闭到托盘 vs 完全退出

**关闭到托盘**（推荐用于长期运行）：
- 点击关闭按钮时，窗口隐藏到系统托盘
- 程序继续在后台运行
- 适合需要长期监控服务器状态的场景

**完全退出**：
- 点击关闭按钮时，程序完全退出
- 所有后台任务停止
- 适合不需要后台运行的场景

### 首次关闭确认

首次点击关闭按钮时，会弹出确认对话框：

1. **选择关闭行为**：
   - ✅ **关闭到托盘**：隐藏到托盘，继续后台运行
   - ❌ **完全退出**：彻底关闭程序

2. **记住选择**（可选）：
   - 勾选「记住我的选择」
   - 下次关闭时将不再询问，直接执行您选择的行为

3. **修改设置**：
   - 如果勾选了「记住我的选择」，想要修改关闭行为
   - 前往「设置」页面重新配置关闭行为

::: warning 注意
如果您选择「完全退出」并勾选了「记住我的选择」，后续点击关闭按钮将直接退出程序，不会显示确认对话框。
:::

## 使用场景

### 场景 1：长期运行服务器监控

**需求**：需要 7×24 小时监控服务器状态，但不希望主窗口一直占用屏幕空间。

**操作**：
1. 启动程序并配置好服务器
2. 点击关闭按钮，选择「关闭到托盘」并勾选「记住我的选择」
3. 程序隐藏到托盘，继续在后台监控服务器
4. 需要查看时，单击托盘图标即可快速打开

**效果**：程序在后台持续运行，定时任务、自动托管等功能正常工作，需要时随时恢复窗口。

### 场景 2：快速访问常用功能

**需求**：需要频繁切换到不同功能页面，但不想每次都打开主窗口。

**操作**：
1. 将程序最小化到托盘
2. 右键点击托盘图标
3. 从托盘菜单直接选择目标功能页面
4. 主窗口自动打开并导航到选定页面

**效果**：无需手动打开窗口再点击导航，一步到位。

### 场景 3：自动托管快速切换

**需求**：需要临时开启或关闭服务端自动更新托管。

**操作**：
1. 右键点击托盘图标
2. 点击「自动托管」菜单项（显示当前状态）
3. 托管状态立即切换

**效果**：无需打开主窗口，直接从托盘菜单切换托管状态，快速高效。

## 常见问题

### 如何从托盘恢复窗口？

有两种方式：
1. **单击托盘图标**：最快捷的方式
2. **右键托盘图标 → 选择「显示窗口」**：适合习惯右键操作的用户

### 如何修改关闭行为？

如果您已经勾选了「记住我的选择」，想要修改关闭行为：
1. 打开主窗口
2. 进入「设置」页面
3. 在「窗口管理」区域重新配置关闭行为
4. 下次关闭时将应用新的设置

### 托盘图标不显示怎么办？

**Windows 11**：
1. 右键点击任务栏空白处 → 任务栏设置
2. 展开「系统托盘图标」
3. 找到「Unturned Server Manager」并启用

**Windows 10**：
1. 点击任务栏右侧的向上箭头（^）
2. 右键点击任务栏空白处 → 任务栏设置
3. 点击「选择哪些图标显示在任务栏上」
4. 找到「Unturned Server Manager」并打开开关

### 最小化到托盘后，服务器还在运行吗？

是的。最小化到托盘只是隐藏了窗口，程序本身仍在后台运行：
- ✅ 服务器继续运行
- ✅ 定时任务继续执行
- ✅ 自动托管继续工作
- ✅ 日志继续记录
- ✅ 系统监控继续更新

### 如何完全退出程序？

有三种方式：
1. **托盘菜单**：右键托盘图标 → 点击「退出」
2. **主窗口**：如果关闭行为设置为「完全退出」，点击关闭按钮
3. **任务管理器**：结束「Unturned Server Manager」进程（不推荐）

::: warning 注意
完全退出程序后，所有后台任务（包括服务器运行、定时任务等）都会停止。
:::
