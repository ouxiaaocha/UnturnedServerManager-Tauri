## 功能概览

| 功能 | 说明 |
| --- | --- |
| 最小化到托盘 | 关闭窗口时隐藏到系统托盘，服务器继续运行 |
| 托盘菜单 | 快速显示/隐藏窗口、导航页面、切换自动托管 |
| 关闭确认 | 首次关闭时选择退出或托盘，并可记住选择 |
| 退出保护 | 如果仍有服务器运行，退出前提示并尝试关闭服务器 |

## 使用建议

长期运行服务器时，建议选择关闭到托盘；真正维护或关机前，再从托盘菜单完全退出。
