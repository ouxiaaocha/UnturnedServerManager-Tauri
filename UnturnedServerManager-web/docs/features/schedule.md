定时任务页面创建和管理服务器的自动重启任务，保障服务器稳定运行。

## 任务类型

### 每日任务

在每天指定时间执行，适合日常维护重启。可设置具体时间（如每天凌晨 4:00）。

### 每周任务

在每周指定日期和时间执行，适合周常维护。

### 间隔任务

每隔指定周期执行，适合高负载服务器。可设置间隔小时数（如每 6 小时）。

## 任务配置

| 配置项 | 说明 |
| --- | --- |
| 任务名称 | 任务的显示名称 |
| 任务类型 | 每日/每周/间隔 |
| 执行时间 | 具体时间或间隔 |
| 启用状态 | 是否启用 |

## 提前提醒

支持在重启前发送提醒消息：

- 提前 N 分钟提醒玩家
- 自定义提醒消息内容
- 通过本地命令 Bridge 广播

::: tip 提示
建议在重启前提醒玩家，避免突然重启丢失游戏进度。
:::

## 使用场景

- **日常维护**：每日凌晨自动重启，清理内存
- **定期更新**：配合服务端更新后自动重启
- **高负载管理**：间隔重启防止内存泄漏

::: warning 数据保存
自动重启会发送提醒并执行保存命令。建议同时开启服务器自动保存。
:::

::: danger 任务冲突
避免过于频繁的重启任务，建议间隔至少 4 小时以上。
:::
