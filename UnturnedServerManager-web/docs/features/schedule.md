## 任务类型

| 类型 | 适合场景 |
| --- | --- |
| 每日任务 | 每天固定时间重启，例如凌晨维护 |
| 每周任务 | 每周固定日期维护，例如周一清晨 |
| 间隔任务 | 每隔若干小时重启一次 |

## 提前提醒

任务可以在重启前发送提醒，让在线玩家有时间保存状态或撤离危险区域。停止和重启优先使用本地命令 Bridge。

::: tip 建议
长期公开服建议配置固定低峰时段重启，并在更新页或工作坊配置里配合自动更新策略。
:::
