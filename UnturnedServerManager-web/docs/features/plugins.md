# 插件管理

插件管理页面用于查看和管理 Unturned 服务器的 Rocket 插件。

## 功能特性

### 插件列表

- 显示 Rocket 插件目录中的所有插件
- 查看插件文件信息
- 快速定位插件配置目录

### 插件备注

为每个插件添加自定义备注：

- 插件名称
- 用途说明
- 版本信息
- 配置说明

### 快速操作

- **打开插件目录**：在文件资源管理器中打开插件所在目录
- **打开配置目录**：快速访问插件的配置文件目录

## Rocket 插件

Unturned 使用 Rocket 模组框架来支持插件系统。插件通常存储在以下目录：

```text
Servers/[存档名]/Rocket/Plugins/
```

### 常见插件类型

| 类型 | 说明 |
| --- | --- |
| 管理插件 | 管理员命令、权限管理 |
| 经济插件 | 游戏内经济系统 |
| 传送插件 | 玩家传送功能 |
| 保护插件 | 领地保护、建筑保护 |
| 信息插件 | 服务器信息显示 |

::: tip 提示
插件备注信息会保存在 `config/` 目录中，不会影响插件本身的配置文件。
:::

## 安装插件

1. 从 Rocket 模组官网或社区下载插件
2. 将插件文件放入 `Rocket/Plugins/` 目录
3. 重启服务器以加载插件
4. 根据需要编辑插件配置文件
