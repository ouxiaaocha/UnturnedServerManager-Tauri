## 可以做什么

| 功能 | 说明 |
| --- | --- |
| 查看插件 | 读取 Rocket/Plugins 目录下的插件文件 |
| 插件备注 | 为插件保存用途、版本、配置注意事项 |
| 打开目录 | 快速打开插件目录或插件配置目录 |
| 联动权限 | 插件需要权限节点时，可到权限标签配置 |

## 使用建议

<div class="doc-card-grid">
  <div class="doc-card"><h3>安装前</h3><p>确认插件兼容 Rocket.Unturned 和当前服务端版本。</p></div>
  <div class="doc-card"><h3>配置时</h3><p>停止服务器后再修改插件配置，避免运行中覆盖。</p></div>
  <div class="doc-card"><h3>排障时</h3><p>查看日志中心和游戏日志，确认插件是否加载失败。</p></div>
</div>
