# RCON 控制台

RCON 控制台允许你通过 Rocket RCON 协议远程管理 Unturned 服务器。当前版本中，服务器页面的启动、停止、重启和本地命令发送默认走本地 Bridge；RCON 保留为额外远程功能。

## 什么是 RCON？

RCON（Remote Console）是一种远程控制协议，允许你通过网络连接到服务器并执行管理命令。Unturned 使用 Rocket RCON 插件来实现这一功能。

如果只在本机管理服务器，建议优先使用服务器页面的本地命令输入栏；它不需要开放 RCON 端口。

## 功能特性

### 连接管理

- 支持手动连接到 RCON 服务器
- 支持服务器启动后自动连接
- 支持断开连接和重新连接

### 命令执行

- 在命令输入框中输入 RCON 命令
- 支持命令历史记录（上下箭头切换）
- 实时显示命令执行结果

### 响应显示

- 显示服务器返回的命令执行结果
- 支持自动滚动到最新响应
- 支持清空响应历史

## 常用 RCON 命令

| 命令 | 说明 |
| --- | --- |
| `say [消息]` | 向所有玩家发送广播消息 |
| `kick [玩家名]` | 踢出指定玩家 |
| `ban [玩家名] [原因] [时长]` | 封禁指定玩家 |
| `unban [玩家名]` | 解封指定玩家 |
| `give [玩家名] [物品ID] [数量]` | 给予玩家物品 |
| `teleport [玩家名] [坐标]` | 传送玩家到指定坐标 |
| `weather [天气]` | 设置天气 |
| `time [时间]` | 设置游戏时间 |

::: tip 提示
使用 RCON 命令前，请确保 Rocket 模块已正确安装，且 RCON 功能已在配置中启用。
:::

## 配置说明

要使用 RCON 功能，需要：

1. 安装 Rocket 模块（可通过首次引导自动安装）
2. 在存档配置中启用 RCON
3. 设置 RCON 端口和密码
4. 确保防火墙允许 RCON 端口的连接

::: tip 提示
RCON 是可选功能，不影响本地启动、停止、重启和本地命令发送。
:::

::: warning 安全建议
RCON 端口不建议对公网开放。建议仅在局域网内使用，或通过 VPN 连接。使用强随机 RCON 密码，并定期更换。
:::
