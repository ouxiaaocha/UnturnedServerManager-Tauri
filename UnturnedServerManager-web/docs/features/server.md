# 服务器控制

服务器控制页面提供对 Unturned 服务器的完整控制能力，包括启动、停止、重启和实时日志查看。

<p align="center">
  <img src="/server.png" width="90%" alt="服务器控制界面" />
</p>

## 核心功能

### 服务器操作

| 操作 | 说明 |
| --- | --- |
| 启动 | 启动 Unturned 服务器进程 |
| 停止 | 发送正常停止信号，等待服务器保存数据后关闭 |
| 重启 | 先停止再启动服务器 |
| 强制停止 | 立即终止服务器进程，不等待数据保存 |

### 实时输出

服务器控制页面实时显示服务器的控制台输出：

- 支持自动滚动到最新日志
- 支持日志搜索功能，快速定位关键信息
- 支持清空当前显示的日志

### 运行模式

支持两种服务器运行模式：

- **局域网模式**：仅局域网内的玩家可以连接
- **互联网模式**：公网玩家可以连接（需要正确配置端口转发）

::: warning 注意
切换运行模式需要重启服务器才能生效。
:::

## 使用场景

### 日常管理

- 查看服务器运行状态
- 启动/停止服务器
- 查看实时日志输出

### 故障排查

- 通过日志搜索定位错误信息
- 查看服务器启动过程中的警告和错误
- 强制停止卡死的服务器进程

### 模式切换

- 在局域网测试和公网部署之间切换
- 根据需要调整服务器运行模式
