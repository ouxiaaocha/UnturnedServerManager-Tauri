## 引导步骤

| 步骤 | 内容 |
| --- | --- |
| SteamCMD | 检测或下载 SteamCMD |
| 服务端目录 | 安装或确认 Unturned Dedicated Server |
| Rocket 模块 | 初始化 Rocket.Unturned 和 Bridge 依赖 |
| 存档初始化 | 创建默认存档和必要配置 |
| RCON 配置 | 生成或填写 RCON 端口与密码 |
