# 首次引导

首次引导页面帮助新用户快速完成 Unturned 服务器的初始配置。

  <img src="/desktop-wizard.png" width="90%" alt="首次引导界面" />
</p>

## 引导步骤

### 1. SteamCMD 安装

首次引导会自动检测和安装 SteamCMD：

- 检测是否已安装 SteamCMD
- 如果未安装，自动下载并解压
- 配置 SteamCMD 路径

### 2. Rocket 模块安装

安装 Rocket 模组框架：

- 从服务端 `Extras` 目录复制 Rocket.Unturned
- 安装到服务端 `Modules` 目录
- 配置 Rocket 模块
- 部署本地命令 Bridge 插件

### 3. 存档初始化

创建和配置服务器存档：

- 创建新的服务器存档
- 配置基础参数（地图、端口、最大玩家数等）
- 生成 Commands.dat 配置文件

### 4. RCON 配置

配置可选的 RCON 远程管理功能：

- 启用 RCON 功能
- 设置 RCON 端口
- 生成随机 RCON 密码

## 自动化配置

首次引导会自动完成以下配置：

| 配置项 | 说明 |
| --- | --- |
| SteamCMD 路径 | 自动检测或下载安装 |
| Rocket 模块 | 自动检测并从服务端 Extras 目录安装 |
| 本地命令 Bridge | 自动部署到当前存档的 Rocket 插件目录 |
| 服务器存档 | 自动创建并初始化 |
| RCON 配置 | 自动生成安全密码 |
| 端口配置 | 自动分配可用端口 |

## 手动配置

如果自动配置失败，可以手动完成：

1. 手动下载 SteamCMD
2. 手动安装 Rocket 模块
3. 在设置页重新部署本地命令 Bridge
4. 手动创建服务器存档
5. 手动配置 RCON 参数

::: tip 提示
首次引导完成后，所有配置都可以在存档配置页面中修改。
:::

## 重新引导

如果需要重新配置服务器，可以：

1. 删除现有存档配置
2. 重新运行首次引导
3. 按照引导步骤重新配置

::: warning 注意
重新引导会覆盖现有配置。请确保已备份重要数据。
:::

## 窄屏适配

首次引导界面针对不同屏幕尺寸进行了优化：

- **桌面端**：完整显示所有配置选项
- **移动端**：简化界面，突出核心操作
- **响应式布局**：自动适应不同屏幕宽度
