# 日志中心

日志中心页面用于查看和管理 Unturned Server Manager 产生的各类日志。

<p align="center">
  <img src="/logs.png" width="90%" alt="日志中心界面" />
</p>

## 日志类型

### 软件日志

记录 Unturned Server Manager 本身的运行日志：

- 应用启动和关闭
- 配置加载和保存
- 错误和异常信息
- 调试信息

### 操作日志

记录用户在软件中的操作：

- 服务器启动/停止操作
- 配置修改记录
- RCON 命令执行
- 定时任务执行

### 游戏日志

记录 Unturned 服务器的游戏日志：

- 玩家连接/断开
- 游戏事件
- 错误和警告信息

## 功能特性

### 日期切换

- 按日期查看历史日志
- 支持快速切换日期
- 自动加载当天日志

### 分类筛选

- 按日志类型筛选
- 按日志级别筛选（信息/警告/错误）
- 支持多条件组合筛选

### 搜索功能

- 关键词搜索
- 实时搜索结果高亮
- 支持正则表达式搜索

## 日志文件位置

日志文件存储在 `logs/` 目录下：

```text
logs/
├── app.log          软件日志
├── operation.log    操作日志
└── game/            游戏日志
    ├── 2024-01-01.log
    ├── 2024-01-02.log
    └── ...
```

## 使用场景

### 故障排查

当服务器出现问题时，通过日志中心快速定位错误原因。

### 安全审计

通过操作日志查看管理员的操作记录。

### 性能监控

通过游戏日志分析服务器性能问题。

::: tip 提示
日志文件会自动按日期分割，避免单个日志文件过大。建议定期清理旧日志以节省磁盘空间。
:::
