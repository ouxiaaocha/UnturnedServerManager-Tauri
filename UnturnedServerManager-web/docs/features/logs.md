日志中心页面查看和管理 Unturned Server Manager 的各类日志。

## 日志类型

### 📱 软件日志

记录应用本身的运行日志：启动/关闭、配置加载/保存、错误异常、调试信息。

### 📝 操作日志

记录用户操作：服务器启停、配置修改、RCON 命令、定时任务执行。

### 🎮 游戏日志

记录 Unturned 服务器日志：玩家连接/断开、游戏事件、错误和警告。

## 功能特性

### 日期切换

- 按日期查看历史日志
- 快速切换日期
- 自动加载当天日志

### 分类筛选

- 按日志类型筛选
- 按级别筛选（信息/警告/错误）
- 多条件组合筛选

### 搜索功能

- 关键词搜索
- 实时高亮结果
- 支持正则表达式

## 日志文件位置

```text
logs/
├── app.log          软件日志
├── operation.log    操作日志
└── game/            游戏日志
    ├── 2024-01-01.log
    └── ...
```

## 使用场景

- **故障排查**：通过日志快速定位错误原因
- **安全审计**：查看管理员操作记录
- **性能监控**：分析服务器性能问题

::: tip 提示
日志文件自动按日期分割。建议定期清理旧日志以节省磁盘空间。
:::
