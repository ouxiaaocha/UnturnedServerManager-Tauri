## 日志类型

| 类型 | 说明 |
| --- | --- |
| 软件日志 | 应用自身错误、后端命令和环境检测信息 |
| 操作日志 | 启停服务器、保存配置、执行更新等用户操作 |
| 游戏日志 | Unturned 服务端输出和插件加载信息 |

## 使用场景

<div class="doc-card-grid">
  <div class="doc-card"><h3>启动失败</h3><p>查看游戏日志和软件日志中的路径、端口、Rocket 加载错误。</p></div>
  <div class="doc-card"><h3>插件异常</h3><p>搜索插件名，确认 DLL 是否加载、依赖是否缺失。</p></div>
  <div class="doc-card"><h3>更新失败</h3><p>查看 SteamCMD 输出，确认网络、磁盘和校验状态。</p></div>
</div>
