# 仪表盘

仪表盘是 Unturned Server Manager 的核心监控页面，提供服务器运行状态的实时概览。

  <img src="/dashboard.png" width="90%" alt="仪表盘界面" />
</p>

## 功能特性

### 实时监控

仪表盘实时显示以下服务器信息：

| 指标 | 说明 |
| --- | --- |
| 运行状态 | 服务器是否正在运行（运行中/已停止） |
| PID | 服务器进程 ID |
| 运行时间 | 服务器已运行的时长 |
| CPU 使用率 | 服务器进程的 CPU 占用百分比 |
| 内存使用 | 服务器占用的内存大小 |
| 网络流量 | 服务器的网络上传/下载流量 |

### 快速操作

仪表盘提供以下快捷操作按钮：

- **启动**：启动 Unturned 服务器
- **停止**：正常停止服务器
- **重启**：重启服务器
- **强制停止**：强制终止服务器进程

::: tip 提示
建议优先使用"停止"而非"强制停止"，以确保服务器数据正确保存。
:::

## 界面布局

仪表盘采用卡片式布局，每个监控指标占据一个独立卡片，方便快速查看：

- 顶部区域显示服务器运行状态和快速操作按钮
- 中部区域显示 CPU、内存、网络等性能指标
- 底部区域显示服务器基本信息（PID、运行时间等）
