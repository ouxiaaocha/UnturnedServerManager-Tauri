## 核心信息

| 区域 | 作用 |
| --- | --- |
| 状态卡片 | 显示当前服务器状态、运行时间和进程 PID |
| 服务器信息 | 展示运行存档、启动模式、连接地址和联机码 |
| 系统监控 | 展示 CPU、内存、上下行流量和总流量 |
| 快捷操作 | 选择存档、切换互联网/局域网模式、启动/停止/重启服务器 |

::: tip 使用建议
仪表盘是日常维护入口。需要修改配置时去「存档」，需要观察完整输出时去「服务器」或「日志」。
:::
