# 便携版说明

## 什么是便携版？

便携版是一个独立的可执行文件，无需安装即可直接运行。程序会在运行目录下自动创建所有必要的数据目录。

```text
unturned-server-manager.exe    ← 直接运行这个文件
```

## 数据目录结构

程序运行时会在 exe 所在目录下自动创建以下数据目录：

```text
config/      应用配置、服务器配置、定时任务、备注数据
logs/        应用日志、操作日志、游戏日志
data/        运行数据
backups/     备份数据
```

### 目录说明

| 目录 | 内容 |
| --- | --- |
| `config/` | 存储应用设置、服务器配置、定时任务定义、模组和插件备注等 |
| `logs/` | 存储应用运行日志、操作审计日志、游戏服务器日志 |
| `data/` | 存储运行时产生的临时数据和缓存 |
| `backups/` | 存储自动和手动创建的配置备份 |

## 使用场景

便携版特别适合以下场景：

- **U 盘携带**：将 exe 文件放在 U 盘中，随时随地管理服务器
- **多服务器管理**：在不同目录放置多个便携版实例，各自管理不同服务器
- **绿色部署**：不需要修改注册表或系统目录，干净整洁
- **快速迁移**：直接复制整个文件夹即可迁移到其他电脑

## 注意事项

::: warning 权限问题
请确保程序运行目录有读写权限。建议不要将程序放在 `C:\Program Files` 等需要管理员权限的目录下。
:::

::: tip 数据备份
便携版的所有数据都存储在 exe 同级目录下，备份时只需复制整个文件夹即可。
:::

::: danger 杀毒软件
部分杀毒软件可能会误报。如果遇到此问题，请将程序目录添加到杀毒软件的白名单中。
:::
