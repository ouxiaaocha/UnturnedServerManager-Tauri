# 🔋 便携版说明

## 什么是便携版？

便携版是独立可执行文件，无需安装即可直接运行。程序在 exe 所在目录下自动创建所有数据目录。

```text
unturned-server-manager.exe    ← 直接运行
```

## 📂 数据目录

运行时自动创建以下目录：

```text
config/      应用配置、服务器配置、定时任务、备注数据
logs/        应用日志、操作日志、游戏日志
data/        运行数据
backups/     备份数据
```

| 目录 | 内容 |
| --- | --- |
| `config/` | 应用设置、服务器配置、定时任务、模组和插件备注 |
| `logs/` | 应用运行日志、操作审计日志、游戏服务器日志 |
| `data/` | 运行时临时数据和缓存 |
| `backups/` | 自动和手动创建的配置备份 |

## 🎯 适用场景

- **U 盘携带**：放在 U 盘中，随时随地管理服务器
- **多服务器管理**：不同目录放置多个实例，各自管理不同服务器
- **绿色部署**：不修改注册表或系统目录，干净整洁
- **快速迁移**：直接复制整个文件夹即可迁移到其他电脑

## ⚠️ 注意事项

::: warning 权限问题
请确保运行目录有读写权限，不要放在 `C:\Program Files` 等需要管理员权限的目录下。
:::

::: tip 数据备份
所有数据都在 exe 同级目录下，备份时只需复制整个文件夹。
:::

::: danger 杀毒软件
部分杀毒软件可能误报，请将程序目录添加到白名单。
:::
